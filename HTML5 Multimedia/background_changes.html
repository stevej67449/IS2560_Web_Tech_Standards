HTML5 Changes

Tag, You're It!

The fundamental change of HTML5 Multimedia is native support. Now it is easy to add video or audio content to a webpage and not rely on users installing third-party plugins to access the elements. To accomplish this HTML5 employs a set of <audio> and <video> tags. These tags are very similar in function to the established <img> tag for adding images to pages.

Benefits

Simplicity is the biggest advantage with this implementation. It is easier to understand from a web development standpoint and, because it has been incorporated into the HTML5 standard, it has good compatibility across different internet browsers.
