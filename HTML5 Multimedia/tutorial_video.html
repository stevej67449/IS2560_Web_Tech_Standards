HTML5 Video Tutorial

Getting Started

Adding video content to a page is easy! Using the video tag you can add a video element much like you would add an image.

Here is some basic code to add a video: 


<video src="video/vid1.mp4" controls='controls'>
   Your browser doesn't support this HTML5 Video.
</video>

This video tag has two attributes, src and controls.
The src attribute defines where the video file is located and can be included in the video tag or a seperate 'source' tag like this:


<video  controls='controls'>
    <source> src="video/vid1.mp4" type='audio/mp4' </source>
   Your browser doesn't support this HTML5 Video.
</video>
The controls attribute adds the bottom control bar to the video. Finally, the text inside the video tag serves as a fallback in case a user's browser doesn't support either HTML5 video or the specific video type used.Here is the end result of that code:

Other Important Attributes

There are many attributes you can use to change the look and behavior of video content in HTML5, but these are a few of the most common attributes used to control video elements. Each attribute includes an example of code, and some also feature a preview of how that code would be implemented.

autoplay

Set as a boolean attribute (true/false).
Allows video to begin play automatically on page load.

Note: In most cases autoplaying is not reccommended
as it can be an annoyance to users or slow down your page. 


<video src="video/vid1.mp4" controls='controls' autoplay='true'>
   Your browser doesn't support HTML5 Video.
</video>

height

Absolute values only, no percentages.
The height of video display. 


<video src="video/vid1.mp4" controls='controls' height=400>
   Your browser doesn't support HTML5 Video.
</video>

loop

Set as a boolean attribute (true/false).
Video will continue to repeat after finishing. 


<video src="video/vid1.mp4" controls='controls' loop='true'>
   Your browser doesn't support HTML5 Video.
</video>

muted

Set as a boolean attribute (true/false).
If set to 'true', video starts without sound.
If set to 'false', sound will play when vieo is played. 


<video src="video/vid1.mp4" controls='controls' muted='true'>
   Your browser doesn't support HTML5 Video.
</video>

width

Absolute values only, no percentages.
The width of video display. 


<video src="video/vid1.mp4" controls='controls' width=50>
   Your browser doesn't support HTML5 Video.
</video>
