Audio Tutorial

Getting Started

Here is some basic HTML code to add audio to a page: 


<audio controls src="audio/audio1.mp3">
   Your browser doesn't support this HTML5 Audio.
</audio>

This audio tag has main two attributes, src and controls.

The src attribute defines where the audio file is located, and can either refrence a location locally or hosted elsewhere on the web.
The controls attribute adds control bar. Additionally, the text inside the audio tag serves as a fallback in case a user's browser doesn't support the specific audio format used.Here is the end result of that code:

Other Important Properties

Unlike video, audio elements do not have features like 'height' or 'width' becasue they lack a visual component and their default controls are not directly editiable without the use of JavaScript. However, developers can change the way that HTML5 audio is presented to the users in a few key ways.

preload

Three possible values.

  1. None: best option to minimize page load traffic.
  2. Metadata: fetches resource metadata like duration.
  3. Auto: allows the entire resource to be downoaded automatically.


<audio controls src="audio/audio1.mp3"  preload="metadata">
   Your browser doesn't support this HTML5 Audio.
</audio>

loop

Adding a loop function will keep audio repeating indefinitely until a user manually sets pause or closes page.
To avoid frustrating users, try not to combine this with autoplay. 


<audio controls src="audio/audio2.mp3"  loop>
   Your browser doesn't support this HTML5 Audio.
</audio>

autoplay

Begin audio track as soon as page is loaded. 


<audio controls src="audio/audio2.mp3"  autoplay='true'>
   Your browser doesn't support this HTML5 Audio.
</audio>

Format Considerations

HTML5 has helped standardize the integration of audio files onto webpages, but there are still some issues developers should be aware of. The most popular audio format on the web is MP3, however due to patent rights not all browsers support this format. When adding audio elements to a site it is especially important to consider duplicating the content in multiple file types for maximum usability Beyond MP3, other popular audio formats include Ogg and Wav.
